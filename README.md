# velocitui

Small, fast-paced, terminal-based arcade game

## Installation

You can install velocitui with [Cargo](https://doc.rust-lang.org/cargo/index.html):

```bash
cargo install --git https://github.com/ellakarhu/velocitui
```

Alternatively, clone the repo and build it:

```bash
git clone https://github.com/ellakarhu/velocitui
cd velocitui
cargo build -r
```

## Usage

run `velocitui -h` for instructions
